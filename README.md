# Express Portes de Garage — Demo (FR/EN)

This is a lightweight bilingual landing page (French first, English second) for Express Portes de Garage.

## How to publish a quick demo link
1. Go to **Netlify Drop**: https://app.netlify.com/drop
2. Drag and drop the ZIP file `express_portes_demo.zip`.
3. Netlify will give you a public URL instantly (e.g., `https://express-portes-demo.netlify.app`).  
   You can paste that link on your Facebook page.

## Edit
- Edit **assets/style.css** for colors/spacing.
- Edit **index.html** for text, phone, and service areas.
